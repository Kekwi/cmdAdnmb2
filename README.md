# cmdAdnmb2
命令行 刷a岛工具
非常简陋的代码 抓包了安卓绿岛
需要安装的库
 ```python
    pip install requests
```

之后运行
```python
    python cmdAdnmb2.py
```
win的也可以直接 下载打包好的exe文件 在命令行里面运行
